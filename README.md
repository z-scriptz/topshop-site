# 🛍️ TopShop — AI-Powered Viral Product Storefront

Welcome to the official frontend repository for **TopShop**, a high-converting, ultra-modern affiliate landing page designed specifically to process and showcase viral products trended on TikTok, Shopee, Amazon, and Pinterest. 

This interface was architected with a cyberpunk-minimalist aesthetic (pitch-black layout with neon accents) to maximize user retention and CTR (Click-Through Rate) coming from mobile social media traffic.

---

## 🚀 Key Features

*   **Mobile-First Design:** Fully responsive layout optimized for seamless navigation on TikTok/Instagram "Link in Bio" traffic.
*   **Jarvis-Ready Architecture:** Designed with structured components to allow future automated CMS product injections via API.
*   **Conversion-Oriented UI:** Sleek glassmorphism product cards with prominent, direct Call-to-Action (CTA) affiliate buttons.
*   **Legal & API Compliance Shield:** Includes robust, standard-compliant Privacy Policy and Terms of Service documents required for official API verifications (YouTube, Meta Graph API, TikTok Shop, and Shopee Enterprise).

---

## 🎨 Tech Stack & Aesthetics

*   **Design Platform:** Framer AI / Carrd.co (Clean-tech component framework)
*   **Palettes:** Pure Black (`#000000`), Electric Purple (`#8A2BE2`), TikTok Cyan (`#00F2FE`)
*   **Typography:** High-readability modern Sans-Serif (Inter / Plus Jakarta Sans)

---

## ⚖️ Compliance & Governance

The `/legal` directory (or footer section) contains the updated **Privacy Policy** and **Terms of Service** (Last updated: June 13, 2026). These documents legally protect the backend automation suite (**Jarvis**), clarifying that the application accesses official platform APIs strictly for private content distribution, automated scheduling, and performance analytics, ensuring zero third-party data leasing.
